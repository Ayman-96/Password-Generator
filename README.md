# Password Generator

A simple interactive Password Generator built with HTML, CSS, and JavaScript.  
It helps users generate strong passwords with customizable options.

---

## Features

- Set password length (8â€“32 characters)  
- Include uppercase letters, numbers, and symbols  
- Allow or prevent duplicate characters  
- Visual password strength indicator  
- Copy password to clipboard easily  

---

## How to Use

1. Open `index.html` in your browser.  
2. Adjust password length using the slider or increase/decrease buttons.  
3. Select options:
   - Uppercase letters
   - Numbers
   - Symbols
   - Duplicate characters allowed
4. Click **Generate Password** to create a password.  
5. Click **Copy to Clipboard** to copy your password.  

---

## Technologies Used

- **HTML** for structure  
- **CSS** for styling  
- **JavaScript** for interactivity  

---

## Notes

- Password strength is indicated by four bars. More bars = stronger password.  
- Duplicate characters can be disabled for higher security.  
- Works fully in modern browsers.
